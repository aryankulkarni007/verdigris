Precedence Table (lowest to highest)
====================================

Level 1: Pipeline
    >>                              left-associative

Level 2: Assignment
    = += -= *= /= %=                right-associative

Level 3: Logical OR
    ||                              left-associative

Level 4: Logical AND
    &&                              left-associative

Level 5: Equality
    == !=                           left-associative

Level 6: Comparison
    < > <= >=                       left-associative

Level 7: Range
    .. ..=                          left-associative

Level 8: Union Type (in types)
    |                               left-associative

Level 9: Additive
    + -                             left-associative

Level 10: Multiplicative
    * / %                           left-associative

Level 11: Cast
    as                              left-associative

Level 12: Unary
    - ! & * (prefix)                right-associative

Level 13: Error Propagation
    ! (postfix)                     left-associative

Level 14: Call / Index / Field
    () [] .                         left-associative

Level 15: Primary
    literals, idents, blocks, if, match, struct lit, array lit, tuple lit
